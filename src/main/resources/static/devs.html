LINKER

Linker API for developers. You can automate cutting process.
For this you should call 'localhost:8080/api' with parametr 'url'.
Method can be 'GET' or 'POST'.

Examples:
js: 
$.ajax({
    url: location.origin + "/api?url=some_url",
    type: "GET",
    success: function (resp) {
        //do something with response
    }
})
